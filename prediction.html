What to predict?

To predict the POPULARITY of TED Talks based on various features extracted from the talk details.

What data to use?


How to define POPULARITY?
Popularity is represented by a categorical number ranging from 1 to 8, determined by the number of views.
As views increase, so does popularity. For instance, 1-9 views correspond to a popularity score of 1, while 10-11 views indicate a score of 2, and so on. With the highest view count recorded at 76,384,176, the maximum popularity score is 8.

Data Cleaning
- Retaining Relevant Information: Retain data attributes crucial for audience access, such as "speaker", "title", "length", and "tag".
- Formatting Data: Transform the date column into numerical features, specifically "year" and "month."
- Eliminating Redundancy: Remove unnecessary columns from the dataset.

Modeling

    Data Preprocessing
    • Transform categorical variables (speaker, title, tag) using techniques like one-hot encoding.
    • Standardize or normalize the length column to ensure consistency in scaling.
    Model Selection
    • Split the data into training, validation, and testing sets to ensure robust model evaluation.
    • Test different models such as Linear Regression, Random Forest, Gradient Boosting, SVR, and Decision Tree.
    • Utilize Grid Search for hyperparameter tuning of each model to optimize performance.
    Model Evaluation
    • Choose the model with the best performance based on cross-validation scores, particularly focusing on neg_mean_squared_error.
    • Compare the models' performance with the best-found hyperparameters to determine the most suitable model for the task.
    Prediction
    • Select the best-performing model and its corresponding hyperparameters.
    • Train this model on the full training dataset to maximize its learning potential.
    • Evaluate the model's performance on the test set using metrics such as Mean Squared Error and R² to assess its predictive capabilities.

Optimization


- The addition of TF-IDF features in Round 2 enhanced the models' performance significantly, indicating the importance of textual data in predicting TED Talks' popularity.
- The incorporation of sentiment analysis in Round 3 did not yield a substantial improvement, slightly reducing the test performance.
- SVR emerged as the best model in Rounds 2 and 3, indicating its effectiveness in handling the complex feature space created by TF-IDF and sentiment analysis.
- Despite enhancements, the overall R² Scores remained low across all rounds, suggesting that other unexplored features or more advanced modeling techniques could be necessary for better prediction accuracy.

This analysis indicates that while text-based features (TF-IDF and sentiment) add value, there is room for further exploration, possibly through more advanced NLP techniques or different modeling approaches, to better capture the factors driving the popularity of TED Talks.
